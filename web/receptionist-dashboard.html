🧑‍💻 RECEPTIONIST DASHBOARD – USER GUIDE

Welcome to the Ocean View Resort Reservation System. Here are the dashboard user guidelines.

🎯 Overview

The Receptionist Dashboard focuses on daily front-desk operations including reservations, guest check-ins, and payments.

1️⃣ Dashboard Home

  • Displays: Available Rooms, Check-ins Today, Ready for Guests status.
  • 📌 Purpose: Quick operational overview.

2️⃣ Room Monitoring

Receptionists can:

  • View room availability and status (Available, Occupied, Maintenance)
  • Search by Room # and filter by Type
  • Update status to Available or Occupied (Cannot edit Maintenance rooms)

3️⃣ Booking Management (Check-ins)

➤ Create New Check-in
  • Click New Check-in -> Select or Add Guest -> Choose Room -> Enter dates -> Save
➤ Manage Check-ins
  • Update status: Confirmed → Checked-in → Completed
  • Cancel check-ins if necessary

4️⃣ Payment Processing

➤ Record Payment
  • Search booking by ID
  • Review bill, apply discounts, select Payment Method (Cash, Card, Online)
  • Click Finalize & Print

⚠ Always confirm payment amount before submission.

🚫 Receptionist Restrictions

  • Receptionists cannot: Add or delete rooms, configure pricing, view revenue analytics, or manage staff (if implemented).

🧾 DAILY WORKFLOW & SECURITY

  • ✔ Check available rooms & create bookings
  • ✔ Check-in guests & process payments to complete booking
  • ✔ Always log out after shift & report errors immediately
