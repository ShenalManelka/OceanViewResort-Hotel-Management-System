🛠 ADMIN DASHBOARD – USER GUIDE

Welcome to the Ocean View Resort Reservation System. Here are the dashboard user guidelines.

🎯 Overview

The Admin Dashboard provides full system control, managing hotel operations, reports, and system monitoring.

1️⃣ Dashboard Home (Overview Panel)

  • Displays: Total Rooms, Available & Occupied Rooms, Maintenance Rooms, Total Revenue.
  • 📌 Purpose: Monitor overall hotel performance in real time.

2️⃣ Room Management

➤ Add Room
  • Navigate to Rooms -> Click Add Room
  • Enter Room Number, Type, Price, Status, and Description
  • Click Save Room
➤ Update & Delete Room
  • Click the Edit icon to update details, or the Trash icon to delete a room.
➤ Room Status Types
  • Available, Occupied, Maintenance
  • ⚠ Admin should ensure room status accuracy at all times.

3️⃣ Booking Management

Admin can: View all reservations, update status, and delete bookings.

Booking Status Flow: Confirmed → Checked-in → Completed (or Cancelled).

4️⃣ Payment Management

Admin can: View payment history, track payment methods, dates, amounts, and statuses across the resort.

5️⃣ Reports & Analytics

Admin has access to: Cancellation Rates, Best Performing Month, Monthly Revenue Chart.

📊 Use this section to analyze business performance.

🔐 Admin Responsibilities

  • ✔ Ensure system accuracy & monitor financial data
  • ✔ Review reports regularly
  • ✔ Maintain room and booking states
